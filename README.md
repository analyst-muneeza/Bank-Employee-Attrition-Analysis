# Bank Employee Attrition Analysis

End-to-End HR Analytics project using Python, SQL, and Power BI to analyze employee attrition trends and identify key factors affecting employee retention.

---

# Project Objective

The objective of this project is to analyze employee attrition patterns and uncover insights that can help organizations improve employee retention strategies and workforce planning.

---

# Tools & Technologies Used

- Python
- SQL (SQLite)
- Power BI
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

# Project Workflow

## 1. Data Collection
- Imported employee attrition dataset
- Verified dataset structure and columns

## 2. Data Cleaning & Preprocessing (Python)
- Handled missing values
- Removed inconsistencies
- Performed feature engineering
- Created cleaned dataset for analysis

## 3. SQL Analysis
- Performed business query analysis
- Analyzed attrition trends
- Used aggregate functions, subqueries, CASE statements, and window functions

## 4. Power BI Dashboard Development
- Built interactive HR analytics dashboard
- Created KPI cards and visual reports
- Added slicers and filters for dynamic analysis

## 5. Business Insights & Recommendations
- Identified high attrition departments and roles
- Generated actionable recommendations for employee retention

---

# KPIs Included

- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary

---

# Key Insights

- Employees aged 36вЂ“45 showed the highest attrition.
- Operations department recorded maximum employee exits.
- Male employees contributed more to attrition.
- Analyst role had the highest attrition among job roles.
- Overtime employees showed slightly higher attrition risk.
- Attrition decreased steadily from 2021 to 2022.

---

# Recommendations

- Improve work-life balance policies.
- Reduce overtime workload where possible.
- Focus retention programs on high-risk departments.
- Conduct employee engagement surveys regularly.
- Create career growth opportunities for employees.
