# Power BI Dashboard

This folder contains the interactive Power BI dashboard created for the Bank Employee Attrition Analysis project.

---

# Dashboard Objective

The dashboard was designed to analyze employee attrition trends and identify key business factors affecting employee retention.

---

# Dashboard Features

- KPI Cards
- Department-wise Attrition Analysis
- Gender-wise Attrition Analysis
- Age-wise Employee Attrition
- Job Role-wise Attrition
- Monthly Attrition Trend
- Overtime vs Attrition Analysis
- Interactive Filters & Slicers
- Summary & Recommendation Page

---

# KPIs Included

- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary

---

# Filters Used

- Gender
- Department
- Age Group

---

# Tools & Technologies

- Power BI
- DAX
- Power Query
- Data Modeling

---

# Key Insights

- Employees aged 36–45 showed the highest attrition.
- Operations department recorded maximum employee exits.
- Analyst role had the highest attrition among job roles.
- Overtime employees showed slightly higher attrition risk.
- Attrition decreased steadily from 2021 to 2022.
