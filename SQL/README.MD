# Bank Employee Attrition Analysis - SQL

This folder contains the SQL analysis performed for the Bank Employee Attrition Analysis project.

---

# Project Objective

The objective of this project is to analyze employee attrition patterns and identify the major factors affecting employee retention using SQL.

---

# Dataset Used

The dataset contains employee-related information such as:

- Employee ID
- Age
- Gender
- Department
- Education Level
- Salary
- Overtime Status
- Job Role
- City
- Attrition Status
- Joining Details

---

# Files Included

- `bank_attrition.db` → SQLite database file
- `bank_attrition_queries.sql` → SQL queries used for analysis
- `cleaned_bank_attrition.csv` → Cleaned dataset used for SQL analysis

---

# SQL Analysis Performed

This project includes 20 SQL business questions ranging from basic to advanced level analysis.

## Key Analysis Areas

- Total employee analysis
- Attrition analysis
- Department-wise attrition
- Overtime impact on attrition
- Job role-wise attrition
- Education-level analysis
- Salary analysis
- City-wise employee distribution
- Attrition trend analysis
- Age-group analysis

---

# Key Insights

- Employees aged 36–45 showed the highest attrition.
- Operations department recorded maximum employee exits.
- Male employees contributed more to attrition.
- Overtime employees showed slightly higher attrition risk.
- Analyst role had the highest attrition among job roles.
- Attrition decreased steadily from 2021 to 2022.
